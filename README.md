# golang-web-static-hello-world

## Description
Serves static html file.

## Tech stack
- bash
- golang 1.13

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
